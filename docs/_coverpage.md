# Rbatis

> Rust ORM Framework Rustlang-based,dynamic sql, no Runtime,No Garbage Collector, low Memory use. support async_std,tokio

> 高性能/易用的Rust首选ORM

* -安全性保障，百分百safe安全代码实现
* -插件支持，分页，逻辑删除，sql拦截器
* -字段支持Format，支持各种数据库函数，类型转换
* -强大易用性，Wrapper解决99%简单sql，py_sql宏解决100%的复杂sql
